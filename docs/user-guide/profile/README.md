# Лоцман — Профиль

Этот раздел описывает личные настройки пользователя. Профиль открывается через меню в правом верхнем углу страницы → **«Профиль»**.

## Содержание раздела

| Документ | Для кого | Прочитайте, если... |
|---|---|---|
| [Размер шрифта интерфейса](font-size.md) | Все пользователи | Хотите сделать текст крупнее или компактнее под свой экран и зрение |

## Что ещё есть в Профиле

Кроме размера шрифта, на странице Профиля доступны:

- **Смена email** и редактирование ФИО — см. [«Управление аккаунтом»](../auth/password-and-totp-management.md).
- **Двухфакторная аутентификация и резервные коды** — см. [«Управление аккаунтом»](../auth/password-and-totp-management.md).
- **Матрица уведомлений** — каналы и режимы доставки.
- **Активные сессии** — просмотр и завершение входов.

## Что важно знать заранее

- **Личные настройки индивидуальны** — ваш выбор виден только вам и не влияет на других пользователей.
- **Размер шрифта переезжает вместе с вами** — он хранится в профиле, поэтому сохраняется при входе с другого устройства или браузера. См. [«Размер шрифта интерфейса»](font-size.md).
