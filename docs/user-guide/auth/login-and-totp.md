# Повседневный вход в Лоцман

Вход в систему занимает несколько секунд. Для него нужны: ваш email, пароль и телефон с приложением-аутентификатором.

---

## Шаги входа

### Шаг 1 из 3. Введите email и пароль

Откройте https://lotsman.example.com. Введите адрес электронной почты и пароль, нажмите «Войти».

### Шаг 2 из 3. Введите код TOTP

```
┌──────────────────────────────────────────┐
│  Шаг 2 из 3: двухфакторная аутентификация│
│                                          │
│  Введите 6-значный код из приложения-    │
│  аутентификатора.                        │
│                                          │
│  Код двухфакторной аутентификации        │
│  [ 123456                              ] │
│                                          │
│  [       Подтвердить код       ]         │
│                                          │
│  [Использовать резервный код]            │
└──────────────────────────────────────────┘
```

Откройте приложение-аутентификатор на телефоне. Найдите запись «Лоцман» (или тот email, который вы указывали при настройке). Введите отображаемый 6-значный код. Код обновляется каждые 30 секунд — если он вот-вот сменится, подождите следующий.

### Шаг 3. Вы в реестре

После успешного ввода кода вы попадаете в реестр документов.

---

## Если код из приложения не принимается — используйте резервный код

Нажмите «Использовать резервный код» под полем кода TOTP.

```
┌──────────────────────────────────────────┐
│  Шаг 2 из 3: резервный код               │
│                                          │
│  Введите один из резервных кодов         │
│  в формате XXXX-XXXX.                    │
│                                          │
│  Резервный код                           │
│  [ A1B2-C3D4                           ] │
│                                          │
│  [    Войти с резервным кодом    ]       │
│                                          │
│  [Вернуться к коду TOTP]                 │
└──────────────────────────────────────────┘
```

Введите один из 10 резервных кодов, которые вы сохранили при первом входе. Формат: `A1B2-C3D4` (8 шестнадцатеричных символов с дефисом посередине).

Каждый резервный код используется **один раз**. После использования он становится недействительным. Если у вас осталось **2 и менее** кодов — система предупредит вас сразу после входа. Перевыпустить резервные коды можно в разделе «Профиль» — см. [Управление аккаунтом](password-and-totp-management.md).

## Сообщение «Неверные учётные данные»

Система намеренно не указывает, что именно введено неверно — пароль, код TOTP или состояние аккаунта. Это сделано в целях безопасности.

Если вы видите это сообщение повторно:
- убедитесь, что Caps Lock выключен,
- убедитесь, что время на телефоне выставлено правильно (TOTP чувствителен к расхождению часов),
- после **5 неверных попыток за 15 минут** аккаунт временно блокируется на 15 минут. Подождите или обратитесь к администратору.

Блокировка идёт по аккаунту, а ошибки пароля и TOTP считаются вместе. Если попытки продолжаются, порог повышается: при **10 неудачных попытках за 60 минут** аккаунт блокируется на **24 часа**, а администратор получает уведомление. В этом случае дождаться снятия блокировки самостоятельно нельзя — обратитесь к администратору.

## Безопасность: не оставляйте сессию открытой на чужом устройстве

Если вы работали на чужом компьютере или в командировке, завершите сессию явно:

1. Нажмите на своё имя или иконку пользователя в шапке страницы.
2. Выберите «Выйти» в меню.

Сессия закрывается немедленно. Если вы забыли выйти, завершить конкретную сессию можно в разделе «Профиль» → «Активные сессии» — см. [Управление аккаунтом](password-and-totp-management.md).
