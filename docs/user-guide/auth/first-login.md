# Первый вход в Лоцман

Этот документ — для тех, кто входит в систему впервые. Вам понадобится одноразовый пароль (OTP), выданный администратором.

---

## Что такое TOTP и зачем он нужен

TOTP — это одноразовый код, который обновляется каждые 30 секунд. Приложение на вашем телефоне генерирует его без подключения к интернету. При каждом входе система проверяет этот код вместе с вашим паролем: даже если пароль попадёт к злоумышленнику, без телефона войти не получится.

Подключение TOTP — обязательный шаг при первом входе. Пропустить его нельзя.

---

## Приложения-аутентификаторы

Выберите одно из следующих (все бесплатны):

- **Google Authenticator** — iOS и Android, простой интерфейс, широко известен.
- **Aegis Authenticator** — только Android, открытый исходный код, поддерживает резервное копирование.
- **FreeOTP** — iOS и Android, разработан Red Hat, минималистичный.

Установите приложение до начала настройки.

## Пошаговая инструкция

### Шаг 1. Получите одноразовый пароль от администратора

Администратор создаёт ваш аккаунт и передаёт вам:
- адрес электронной почты (он же логин),
- одноразовый пароль (OTP) — передаётся лично или через Telegram.

Одноразовый пароль действует **10 минут** и может быть использован **один раз**.

### Шаг 2. Откройте Лоцман

Перейдите по адресу вашей установки — его сообщит администратор. Например:

```
https://lotsman.example.com
```

Вы увидите экран входа с полями «Электронная почта» и «Пароль».

```
┌─────────────────────────────────────────┐
│  Лоцман — Реестр документов             │
│  Войдите в Лоцман для доступа к реестру │
│                                         │
│  Электронная почта                      │
│  [ alice@company.ru                   ] │
│                                         │
│  Пароль                                 │
│  [ ••••••••                           ] │
│                                         │
│  [          Войти          ]            │
└─────────────────────────────────────────┘
```

### Шаг 3. Введите email и одноразовый пароль

В поле «Электронная почта» введите ваш рабочий email. В поле «Пароль» введите одноразовый пароль, выданный администратором. Нажмите «Войти».

Система определит, что это первый вход, и перейдёт к настройке двухфакторной аутентификации.

### Шаг 4. Отсканируйте QR-код

На экране появится QR-код. Откройте приложение-аутентификатор и отсканируйте его.

```
┌──────────────────────────────────────────┐
│   (1) ── (2) ── (3)                      │
│   Вход   TOTP   Пароль                   │
│                                          │
│  Отсканируйте QR-код в приложении        │
│  (Google Authenticator, FreeOTP, Aegis)  │
│                                          │
│   ┌──────────────┐                       │
│   │ ░░▓▓░░▓░░░▓▓ │  ← QR-код            │
│   │ ▓░▓░▓▓░▓░▓░░ │                       │
│   │ ░▓░░░▓░░░▓▓░ │                       │
│   └──────────────┘                       │
│                                          │
│  [Ввести ключ вручную]                   │
└──────────────────────────────────────────┘
```

Если QR-код не сканируется, нажмите «Ввести ключ вручную» — система покажет текстовый ключ в формате base32 (например, `JBSWY3DPEHPK3PXP`). Введите его в приложение самостоятельно.

### Шаг 5. Введите 6-значный код из приложения

После сканирования приложение начнёт показывать коды, меняющиеся каждые 30 секунд. Введите текущий код в поле «Подтвердите код из приложения» и нажмите «Подтвердить».

Если код не принимается — проверьте, что часы телефона выставлены автоматически. TOTP завязан на время, и расхождение в минуту делает коды неверными.

### Шаг 6. Сохраните резервные коды

Сразу после подтверждения TOTP система покажет **10 резервных кодов**.

```
┌─────────────────────────────────────────────────┐
│  Сохраните эти коды в надёжном месте!            │
│  Они будут показаны только один раз.             │
│                                                 │
│  A1B2-C3D4    E5F6-7890    AB12-CD34            │
│  EF56-7890    1234-ABCD    5678-EF01            │
│  23AB-45CD    67EF-8901    ABCD-1234            │
│  5678-90EF                                      │
│                                                 │
│  [ ] Я сохранил резервные коды в надёжном месте │
│                                                 │
│  [              Продолжить              ]        │
└─────────────────────────────────────────────────┘
```

> **Важно.** Каждый код используется **один раз**. После использования он становится недействительным. Если вы потеряете телефон и резервные коды одновременно, только администратор сможет восстановить доступ.

Куда сохранить коды:
- в менеджер паролей (KeePass, Bitwarden и т. п.),
- распечатать и хранить в надёжном месте.

Поставьте галочку «Я сохранил резервные коды» и нажмите «Продолжить».

### Шаг 7. Задайте постоянный пароль

На последнем шаге введите новый пароль для вашего аккаунта.

Требования к паролю:
- минимум **12 символов** (максимум 1024),
- нет ограничений на символы — используйте парольный менеджер,
- нельзя использовать пароли из известных баз утечек.

После нажатия «Установить пароль» вы попадёте в реестр документов.

---

## Что делать, если QR-код не сканируется

1. Нажмите «Ввести ключ вручную» под QR-кодом.
2. Скопируйте текстовый ключ (base32-строка, например `JBSWY3DPEHPK3PXP`).
3. В приложении-аутентификаторе выберите «Добавить аккаунт вручную» → введите email и ключ.
4. Вернитесь в браузер и введите 6-значный код из приложения.
