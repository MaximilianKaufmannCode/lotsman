## Прочитайте в первую очередь, если...

- **Первый вход** — начните с [«Первый вход»](first-login.md). Вам понадобится одноразовый пароль от администратора.
- **Забыли пароль** — у Лоцмана нет самостоятельного сброса пароля. Обратитесь к администратору, он выдаст новый одноразовый пароль.
- **Потеряли телефон или удалили приложение TOTP** — обратитесь к администратору. Он сбросит настройку двухфакторной аутентификации, и вы сможете подключить новое устройство.
- **Вы администратор** — начните с [«Руководства администратора»](admin-guide.md).

## Как устроена защита входа

Лоцман использует двухэтапную проверку при каждом входе:

1. **Пароль** — известен только вам.
2. **Код TOTP** — одноразовый 6-значный код из приложения-аутентификатора на вашем телефоне.

Оба шага обязательны для всех пользователей, включая администраторов.
