# Поиск и фильтры

Лоцман выполняет поиск на стороне сервера: при каждом изменении фильтра запрос уходит в базу данных, а результат возвращается за доли секунды. Фильтры сохраняются в URL — ссылкой можно поделиться.

## Глобальный поиск (⌘K / Ctrl+K)

Нажмите **⌘K** (macOS) или **Ctrl+K** (Windows/Linux) либо кликните на строку поиска в верхней части страницы.

Поиск охватывает:
- название компании
- номер документа
- тип документа
- заметки

Запрос отправляется после **200 мс** паузы (дебаунс), чтобы не перегружать сервер при быстром наборе. При наборе менее 2 символов запрос не отправляется, а под строкой поиска появляется подсказка «Введите минимум 2 символа».

Поиск использует полнотекстовый индекс `pg_trgm` — он находит частичные совпадения внутри слов. Запрос «Газпром» найдёт «ООО Газпром Инвест» и «АО Газпромнефть».

## Фильтры по колонкам

Над каждой колонкой есть выпадающий фильтр:

| Фильтр | Поведение |
|---|---|
| **Тип документа** | Выберите один тип из списка; в таблице останутся только документы этого типа. |
| **Статус** | Выберите «ОК», «Скоро», «Просрочено» или «В архиве». |
| **Компания** | Выберите конкретную компанию из списка. |

Все фильтры работают одновременно и суммируются: например, «Тип = Лицензия» + «Статус = Просрочено» покажет только просроченные лицензии.

## Сортировка

Кликните на заголовок любой колонки, чтобы отсортировать по возрастанию. Второй клик — по убыванию. Значок стрелки показывает текущее направление.

Для вторичной сортировки (например, сначала по типу, затем по дате) удерживайте **Shift** и кликните второй заголовок.

NULL-значения (например, пустая дата истечения) при сортировке оказываются в конце.

## Фильтры в URL

Активные фильтры и сортировка автоматически отражаются в адресной строке браузера. Пример:

```
/registry?q=Газпром&type_code=contract&status=soon&sort=expiry_date&dir=asc
```

Скопируйте URL и откройте его в новой вкладке или отправьте коллеге — фильтры применятся автоматически.

## Сброс фильтров

Нажмите кнопку **«Сбросить фильтры»** в тулбаре, чтобы вернуться к полному списку активных документов. Эта кнопка появляется только если хотя бы один фильтр активен.

Если поиск не дал результатов, таблица показывает сообщение «Ничего не найдено» с кнопкой «Сбросить фильтры».
