# Лоцман — Аутентификация и безопасность аккаунта

Этот раздел описывает всё, что связано с входом в систему, защитой аккаунта и управлением пользователями.

## Содержание раздела

| [Руководство администратора](admin-guide.md) | Администратор | Вы создаёте пользователя, сбрасываете пароль или блокируете аккаунт |

## Что важно знать заранее

- **Забыли пароль** — самостоятельного сброса нет. Обратитесь к администратору, он выдаст новый одноразовый пароль.
- **Потеряли телефон или удалили приложение TOTP** — тоже к администратору: он сбросит двухфакторную аутентификацию, и вы подключите новое устройство.
- **Личные настройки** (смена email, размер шрифта, уведомления) — в разделе [«Управление аккаунтом»](password-and-totp-management.md), вкладка «Профиль».

## Как устроена защита входа

Лоцман использует двухэтапную проверку при каждом входе:

1. **Пароль** — известен только вам.
2. **Код TOTP** — одноразовый 6-значный код из приложения-аутентификатора на вашем телефоне.

Оба шага обязательны для всех пользователей, включая администраторов.
