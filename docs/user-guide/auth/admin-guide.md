# Руководство администратора

Этот документ — для пользователей с ролью «Администратор». Управление пользователями доступно через раздел «Пользователи» в навигационном меню (виден только администраторам).

Все чувствительные действия требуют подтверждения вашим текущим кодом TOTP.

---

## Создание нового пользователя

1. Откройте раздел «Пользователи» → нажмите «Добавить пользователя».
2. Введите ваш текущий код TOTP для подтверждения.
3. Заполните форму:
   - **ФИО** — полное имя сотрудника,
   - **Электронная почта** — рабочий email (будет логином),
   - **Роль** — `Администратор`, `Редактор` или `Читатель`.
4. Нажмите «Создать».
5. Система отобразит **одноразовый пароль (OTP)** — он показывается **один раз**.

Передайте OTP пользователю лично или через защищённый канал (например, Telegram). После закрытия окна получить его повторно невозможно. Пользователь должен войти в течение 10 минут и пройти настройку.

---

## Сброс пароля сотрудника

Используйте этот инструмент, если сотрудник забыл пароль или заблокирован.

1. В разделе «Пользователи» найдите нужного пользователя.
2. В меню действий выберите «Сбросить пароль».
3. Введите ваш текущий код TOTP для подтверждения.
4. Система сгенерирует новый одноразовый пароль и завершит все активные сессии пользователя.
5. Одноразовый пароль показывается **один раз** — скопируйте и передайте сотруднику.

Сотрудник войдёт с этим OTP и будет обязан установить новый постоянный пароль перед доступом к реестру.

---

## Сброс TOTP сотрудника

Используйте этот инструмент, если сотрудник потерял телефон или удалил приложение-аутентификатор.

1. В меню действий для пользователя выберите «Сбросить TOTP».
2. Введите ваш текущий код TOTP для подтверждения.
3. Нажмите «Сбросить TOTP».

Что происходит после сброса:
- все активные сессии пользователя завершаются немедленно,
- при следующем входе пользователь будет направлен на повторную настройку TOTP.

Пользователю потребуется новый одноразовый пароль. Если его нет — сначала выполните «Сброс пароля» (см. выше).

---

## Блокировка пользователя

Мгновенная блокировка — подходит в экстренных случаях (например, компрометация аккаунта).

1. В меню действий выберите «Заблокировать».
2. Введите ваш текущий код TOTP для подтверждения.

Блокировка действует немедленно: все сессии пользователя завершаются, новые входы невозможны. Блокировка не имеет срока действия — разблокировать можно только вручную через «Разблокировать» в том же меню.

---

## Просмотр сессий пользователя

1. В меню действий выберите «Завершить все сессии» или откройте карточку пользователя.
2. Чтобы просмотреть активные сессии любого пользователя — это доступно через раздел управления пользователями.

Список сессий показывает устройство, IP-адрес и дату создания каждой сессии.

---

## Изменение роли

1. В меню действий выберите «Сменить роль».
2. Введите ваш текущий код TOTP для подтверждения.
3. Выберите новую роль: `Администратор`, `Редактор` или `Читатель`.
4. Нажмите «Сменить роль».

Роли в системе:
- **Администратор** — полный доступ, включая управление пользователями.
- **Редактор** — просмотр и редактирование реестра документов.
- **Читатель** — только просмотр реестра документов.

---

## Деактивация аккаунта

Деактивация — это мягкое удаление: аккаунт закрывается, но данные сохраняются в системе.

1. В меню действий выберите «Деактивировать».
2. Введите ваш текущий код TOTP для подтверждения.
3. Подтвердите действие.

После деактивации все сессии пользователя завершаются немедленно, вход становится невозможным. Деактивация необратима через интерфейс — для восстановления аккаунта обратитесь к технической поддержке.
