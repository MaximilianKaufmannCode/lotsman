# Размер шрифта интерфейса

Лоцман позволяет выбрать комфортный размер шрифта всего интерфейса. Настройка хранится в вашем профиле и применяется мгновенно — без перезагрузки страницы.

## Где находится

Откройте меню в правом верхнем углу страницы → **«Профиль»**. Раздел **«Размер шрифта интерфейса»** — второй блок на странице, сразу под основными данными аккаунта.

## Четыре размера

Выберите один из четырёх пресетов в переключателе:

| Пресет | Когда подходит |
|---|---|
| **Компактный** | Хотите видеть больше строк реестра на экране |
| **Обычный** | Размер по умолчанию |
| **Крупный** | Текст комфортнее читать без наклона к экрану |
| **Очень крупный** | Максимальная читаемость; подходит для больших мониторов и слабого зрения |

Над названием каждого пресета показана буква **А** соответствующего размера — наглядная подсказка.

## Мгновенное применение и живой предпросмотр

- Размер меняется **сразу при выборе** — весь интерфейс (заголовки, таблица реестра, кнопки) перестраивается без перезагрузки.
- Под переключателем есть блок **«Предпросмотр»** с примером текста интерфейса — он меняется вместе с выбранным размером, чтобы вы оценили результат до того, как закроете страницу.

## Сброс к размеру по умолчанию

Если выбран размер, отличный от обычного, под предпросмотром появляется кнопка **«Сбросить по умолчанию»**. Она возвращает размер к пресету «Обычный» одним нажатием.

## Настройка переезжает вместе с вами

Размер шрифта сохраняется **в вашем профиле на сервере**, а не только в текущем браузере. Это значит:

- вы войдёте с другого компьютера, ноутбука или из другого браузера — и увидите тот же выбранный размер;
- переустановка браузера или очистка его данных настройку не сбросит.

> Настройка индивидуальна: ваш выбор виден только вам и не влияет на других пользователей.

## Совместимость с зумом браузера (доступность)

Размер шрифта Лоцмана и **масштаб браузера** (Ctrl + «+» / Ctrl + «–») работают независимо и складываются. Можно комбинировать оба способа: например, выбрать «Крупный» в Профиле и дополнительно увеличить масштаб страницы в браузере.

Интерфейс построен на относительных единицах, поэтому корректно масштабируется и не «ломается» при увеличении — это соответствует требованиям доступности WCAG к изменению размера текста.

## Клавиатурное управление

Переключатель размеров полностью управляется с клавиатуры:

1. Перейдите к переключателю клавишей **Tab**.
2. Переключайте пресеты стрелками — **←/→** или **↑/↓**.
3. **Home** выбирает первый пресет (Компактный), **End** — последний (Очень крупный).

Размер применяется сразу при перемещении по пресетам, а программы экранного доступа озвучивают выбранный размер.
