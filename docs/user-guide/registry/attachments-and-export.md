# Вложения и экспорт

## Вложения

### Прикрепить файл к документу

1. Кликните на строку документа, чтобы открыть боковую панель.
2. Перейдите на вкладку **«Вложения»**.
3. Перетащите файл в область вкладки **или** нажмите **«Добавить файл»** и выберите файл в проводнике.

Для прикрепления файлов нужна роль **редактора** или **администратора**.

### Допустимые типы файлов

| Тип | MIME-тип |
|---|---|
| PDF | `application/pdf` |
| JPEG | `image/jpeg` |
| PNG | `image/png` |
| TIFF | `image/tiff` |
| DOCX (Word) | `application/vnd.openxmlformats-officedocument.wordprocessingml.document` |
| XLSX (Excel) | `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` |

Тип файла проверяется по содержимому, а не по расширению. Файл с расширением `.pdf`, который на самом деле является другим форматом, будет отклонён.

### Ограничения

- **Максимальный размер:** 25 МиБ на файл. При превышении появляется ошибка «Файл больше 25 МиБ».
- **Нельзя прикрепить файл к архивному документу** — сначала восстановите документ (только администратор).

### Скачать файл

В списке вложений нажмите на имя файла. Браузер начнёт загрузку. Ссылка формируется в момент клика и действует **менее 60 секунд** — это сделано намеренно, чтобы нельзя было передать прямую ссылку на файл третьим лицам.

Скачать вложение может любой аутентифицированный пользователь, включая читателей. Файлы архивных документов также остаются доступны.

### Удалить файл

В списке вложений нажмите значок удаления рядом с именем файла. Удаление необратимо — файл удаляется физически. Операция доступна редакторам и администраторам. Нельзя удалить вложение из архивного документа.

## Экспорт в .xlsx

### Запустить экспорт

1. Настройте нужные фильтры и сортировку в реестре.
2. Нажмите **«Экспорт .xlsx»** в тулбаре.

Система запускает формирование файла в фоне. Появится уведомление «Экспорт запущен. Вы получите уведомление о готовности». Файл отражает состояние реестра **в момент нажатия кнопки** — последующие изменения в реестре в уже запущенный экспорт не попадут.

Вы можете продолжать работу в реестре пока файл готовится. Имя файла: `Лоцман_реестр_ГГГГ-ММ-ДД.xlsx`.

### Проверить статус и скачать

1. Нажмите иконку экспортов (или дождитесь уведомления о готовности).
2. В модалке «Экспорты» найдите нужный экспорт.
3. Нажмите **«Скачать»**.

Файл доступен для скачивания в течение **24 часов** после завершения формирования.

### Статусы экспорта

| Статус | Что происходит |
|---|---|
| **Формируется** | Задача выполняется. Подождите. |
| **Готов** | Файл готов. Нажмите «Скачать». |
| **Ошибка** | Формирование не удалось. Нажмите «Экспорт .xlsx» повторно. |
| **Истёк** | Прошло больше 24 часов. Запросите экспорт заново. |

Если статус «Истёк» — просто нажмите «Экспорт .xlsx» снова с теми же фильтрами. При ошибке «Файл экспорта истёк. Создайте новый экспорт» сделайте то же самое.
