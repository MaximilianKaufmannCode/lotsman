# Лоцман — Каналы уведомлений и приглашения

Этот раздел — для пользователей с ролью **«Администратор»** (Space-admin).

---

## Для кого этот раздел

У Вас есть доступ к разделу `/admin` в веб-интерфейсе. Именно Вы настраиваете каналы уведомлений (Email, Telegram, Dion, Календарь Exchange, ICS-подписка), приглашаете коллег и следите за тем, чтобы в системе было не менее двух администраторов.

Если Вы не видите меню **«Администрирование»** — значит Ваша роль не `admin`. Обратитесь к другому администратору.

## Что НЕ входит в этот раздел

Следующие операции требуют прямого доступа к серверу (SSH) и выполняются только системным администратором (super-admin, DevOps):

- Генерация и ротация ключей шифрования (`CHANNEL_ENC_KEY`, `TOTP_ENC_KEY`).
- Восстановление единственного администратора, если он потерял пароль и TOTP одновременно.
- Восстановление зашифрованных конфигураций каналов после утраты ключа.
- Развёртывание, обновление и бэкап.

По этим вопросам обращайтесь к сопровождающему инстанс Лоцмана; см. [руководство по эксплуатации](../../deployment/preprod-runbook.md).
